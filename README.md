# Be a Caricature of Your Own

A fun SaaS web app where users capture a selfie, share hobbies & future plans, and get a hilarious personalized caricature generated by Grok Imagine (xAI).

## Features
- Webcam capture or image upload
- Personalized prompts based on user input
- Grok Imagine API integration for image editing
- Modern responsive UI

## Quick Start

### Backend
```bash
cd backend
cp .env.example .env
# Add your XAI_API_KEY
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Next.js frontend runs on `http://localhost:3001`.
Express backend runs on `http://localhost:3000`.

## Deployment
- Frontend: Vercel / Netlify
- Backend: Render / Railway / Fly.io
- Add environment variables for production.

## Tech Stack
- Frontend: Next.js, React, Tailwind CSS
- Backend: Node.js, Express, Multer, Axios
- AI: xAI Grok Imagine API
